# 对比/选型报告模板

> 适用：技术选型、方案对比、供应商评估、产品对比

## 摘要

> 推荐方案 + 关键理由

## 目标

> 选型目标、约束、评估维度

## 候选方案

> 2-4 个候选，简述各自定位

## 多维对比

> 表格/矩阵呈现，至少 5 维

| 维度 | 方案 A | 方案 B | 方案 C |
|------|--------|--------|--------|
| 成本 | | | |
| 性能 | | | |
| 易用性 | | | |
| 生态 | | | |
| 风险 | | | |

## 详细分析

> 每个方案的优势/劣势/适用场景

## 推荐

> 首选方案 + 备选 + 决策理由

## 实施路径

> 落地步骤、里程碑、风险控制
