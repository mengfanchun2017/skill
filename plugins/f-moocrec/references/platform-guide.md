# MOOC 平台对比指南

## 平台总览

| 维度 | Coursera | edX | MIT OCW | 中国大学MOOC |
|------|----------|-----|---------|-------------|
| 课程数 | 7,000+ | 4,600+ | 2,400+ | 10,000+ |
| 大学合作 | 275+ | 260+ | MIT only | 700+ 中国高校 |
| 免费模式 | ~300 门完全免费，其余仅预览首个模块 | 大部分可免费旁听全部内容 | 全部免费 | 全部免费 |
| 证书价格 | $49-79/课 或 $399/年 Plus | $50-300/课 | 无证书 | 免费/少量认证费 |
| 证书验证 | Credly 徽章 + 区块链 | 唯一验证 URL | — | icourse163.org |
| 语言 | 主要为英语，部分有中字 | 主要为英语 | 英语 | 中文 |
| 学位项目 | 有（$9K+） | 有（$10K+） | — | — |
| 适合人群 | 职业导向，IT/数据/商业 | 学术导向，科学/工程/人文 | 自学，不计证书 | 中文环境，国内认可 |

## 详细分析

### Coursera

**2026 现状**: 合并 Udemy 后课程量翻倍。免费模式大幅缩水——大部分课程只有首个模块免费预览。

**优点**:
- Specialization 体系完整（多门课组成专项）
- Credly 数字徽章标准化，LinkedIn 支持最佳
- 移动端体验好
- 中文课程不断增加

**缺点**:
- 免费访问大幅受限（2025 年改版后）
- 价格透明度不如 edX
- 学术深度有时不如 edX

**定价**:
- 单课: $49-79
- Coursera Plus 月度: $59/月
- Coursera Plus 年度: $399/年（最划算）
- 学位: $9K-25K

**生物科学优势领域**: Bioinformatics, Genomics, Cancer Biology, Neuroscience

### edX

**2026 现状**: 被 2U 收购后仍在运营，课程质量保持高位。大部分课程仍可免费旁听。

**优点**:
- 免费旁听最慷慨（绝大部分课程全部内容可免费学）
- Harvard/MIT 原生课程最多
- 证书含大学 logo，背书效果好
- MicroMasters 可转学分

**缺点**:
- 课程量不如 Coursera
- 平台体验略旧（被收购后仍在改进）
- 价格略高（$50-300/课）

**定价**:
- 免费旁听: $0
- Verified Track: $50-300/课
- MicroMasters: $600-1,500
- 学位: $10K-25K

**生物科学优势领域**: Introductory Biology, Biochemistry, Genetics, Cell Biology, Neuroscience, Immunology

### MIT OpenCourseWare (OCW)

**2026 现状**: 持续更新，仍是最完整的免费大学课程资源。

**优点**:
- 完全免费，无付费墙
- 含完整视频、讲义、习题、考试
- Gilbert Strang 线代、Eric Lander 生物等传奇课程
- 不可替代的经典课程（18.06 线代、7.013 生物）

**缺点**:
- 无证书
- 无互动/社区
- 部分课程年代较久
- 无移动端

**生物科学经典课程**:
- 7.013 Introductory Biology
- 7.012 Introduction to Biology  
- 5.111 Principles of Chemical Science
- 18.06 Linear Algebra (Gilbert Strang)

### 中国大学MOOC (icourse163.org)

**2026 现状**: 中国最大的 MOOC 平台，700+ 合作高校。

**优点**:
- 中文授课，无语言障碍
- 全部免费
- 有国家级精品课程
- 与国内高校课程体系对齐

**缺点**:
- 国际认可度有限
- 课程深度参差不齐
- 缺少互动/讨论
- 证书对国外升学/求职助力有限

**推荐课程**:
- 北京大学：生物化学
- 武汉大学：遗传学
- 华中农业大学：分子生物学

## 双轨策略

对每门核心课，给学生两条路：

```
免费轨道                      证书轨道
┌─────────────────┐          ┌─────────────────┐
│ edX 免费旁听      │          │ edX Verified     │
│ - 全部视频+习题   │          │ - 全部视频+习题   │
│ - 无证书          │          │ - 考试+评分       │
│ - 无截止日期      │          │ - Harvard/MIT logo│
│ - $0             │          │ - 验证 URL        │
│                  │          │ - $50-300         │
└─────────────────┘          └─────────────────┘
```

学生可以先免费旁听 → 觉得值得再升级拿证。edX 支持中途升级（已完成的作业成绩保留）。

## 推荐组合

| 课程类型 | 推荐平台 | 原因 |
|---------|---------|------|
| 数理基础 | MIT OCW | 完全免费，经典课程 |
| 核心生物 | edX (free audit) | Harvard/MIT 原生，免费旁听 |
| 进阶专题 | Coursera + Credly | 证书体系好，可挂 LinkedIn |
| 中文补充 | 中国大学MOOC | 中文环境，基础更扎实 |
